### 5.2 进阶

#### [23. 合并 K 个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/description/)
 
给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

##### 优先队列

```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * 合并K个升序链表
 * @param {ListNode[]} lists - 包含K个升序链表的数组
 * @return {ListNode} - 合并后的新链表头节点
 * @description 
 * 1. 使用最小堆优化比较过程
 * 2. 时间复杂度：O(n log k) n为总节点数，k为链表数
 * 3. 空间复杂度：O(k) 堆存储空间
 * @note 当链表数组包含空链表时自动跳过
 */
var mergeKLists = function(lists) {
    // 初始化最小堆（按节点值排序）
    const pq = new MinPriorityQueue(q => q.val);
    
    // 将非空链表头节点入堆
    for (let list of lists) {
        if (list) pq.enqueue(list);
    }
    
    // 创建哑节点作为结果链表起点
    let dummy = new ListNode();
    let curr = dummy;
    
    // 循环处理堆中元素
    while (pq.size()){
        // 取出当前最小节点
        curr.next = pq.dequeue();
        curr = curr.next;
        
        // 将取出节点的后续节点入堆
        if (curr.next) {
            pq.enqueue(curr.next);
        }
    }
    return dummy.next;
};

```
