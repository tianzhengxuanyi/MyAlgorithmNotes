### 冒泡排序

**时间复杂度：** `O(N^2)`

**空间复杂度：** `O(1)`

**稳定性：** 稳定

**算法步骤：**

- 遍历数组待排序部分
- 比较当前位置和相邻下个位置数字大小，如果当前位置大，交换两个位置的数
- 遍历完成后，待排序部分最右侧的值为待排序部分的最大值，待排序长度-1
- 重复上述步骤

```js
function bubbleSort(arr) {
  if (arr === null || arr.length < 2) {
    return;
  }
  for (let i = arr.length - 1; i > 0; i--) {
    for (let j = 0; j < i; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
}
```
