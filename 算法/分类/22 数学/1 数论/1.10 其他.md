#### [633. 平方数之和](https://leetcode.cn/problems/sum-of-square-numbers/description/)

给定一个非负整数  `c` ，你要判断是否存在两个整数 `a` 和 `b`，使得  `a2 + b2 = c` 。

**示例 1：**

```
输入：c = 5
输出：true
解释：1 * 1 + 2 * 2 = 5
```

**示例 2：**

```
输入：c = 3
输出：false
```

**提示：**

- `0 <= c <= 231 - 1`

##### 双指针

```js
/**
 * @param {number} c
 * @return {boolean}
 */
var judgeSquareSum = function (c) {
  let left = 0,
    right = Math.floor(Math.sqrt(c));
  while (left <= right) {
    let t = left * left + right * right;
    if (t == c) {
      return true;
    } else if (t > c) {
      right--;
    } else {
      left++;
    }
  }

  return false;
};
```
